# Attributes

namedefaultdesc
title弹框标题
value/v-model是否显示弹框，支持双向绑定
width46%弹框宽度
height400弹框高度
top12%弹框y坐标
left弹框x坐标 默认居中显示
zIndex100弹框层级
minWidth480最小宽度
minHeight300最小高度
handles['br']可拉伸的位置
draggabletrue是否可拖拽
resizabletrue是否可拉伸
showMasktrue是否下那是mask
appendToBodytrue是否添加到body下
